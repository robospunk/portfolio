Services

Whether you are a blogger, business owner, or independent artist

I can make what you’re looking for

Dynamic Sites

If you'd like to be able to manage content on your site, this is what you want. For sites like this, I normally use Wordpress, which is an easy to use content management system. This is really good for blogs, galleries, and online stores.

Static Sites

If you don't need to change the content on your site very often, this is usually the way to go. Sites like this, I usually code from scratch. The benefit of static sites is that they can be designed around the content, unlike a dynamic site.

Web Design

Maybe you already know how to code websites but don't know how to make it pretty. I can do that! How do you want your website to feel? I can figure out the layout and color palette. I also really like to make animated icons if that's all you want.

About Joey

I’ve been creating for as long as I can remember. As a young child I was infatuated with drawing and making flip-book animations with post-it notes. In highschool I delved deeper into graphic design, animation and video editing.

I operate a small pinback button business, make webcomics, and do freelance web development. I currently live in Western Massachusetts with my girlfriend and our two cats.
